I am building an IoT weather monitoring and prediction system using Arduino, Python, Flask, and Scikit‑learn.

Sensor readings are collected from an Arduino board and sent to my Python application through Serial Communication.

Hardware setup:
- Arduino Uno connected via USB
- Sensors used:
  • DHT11/DHT22 – temperature and humidity
  • Rain sensor – precipitation
  • UV sensor – UV index
  • BMP180/BMP280 – atmospheric pressure

Arduino sends sensor readings in CSV format through the serial port.

Example format:
temperature,humidity,precipitation,uv_index,pressure

Example data:
24.5,65.3,0.0,3.2,1012.8

Python reads this using pyserial:

serial.Serial('COM3', 9600)

The system then uses machine learning models to:
1. Classify the current weather
2. Predict weather for the next 2 hours

Models used:
- weather_model.pkl
- forecast_model.pkl
- label_encoder.pkl
- weather_scaler.pkl
- forecast_encoder.pkl
- forecast_scaler.pkl
